O valor inherit no CSS

Como forçar a herança de propriedades CSS.

Exemplo Visual

Pai (cor: azul)
Filho (cor herdada do pai)
Filho (cor forçada para herdar)

Resumo

  • inherit força a propriedade a herdar o valor do elemento pai.
  • Útil quando a propriedade normalmente não é herdada.
