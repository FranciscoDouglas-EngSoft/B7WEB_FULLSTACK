A base geral do CSS

Entenda a estrutura e funcionamento do CSS.

Sintaxe do CSS

seletor {
    propriedade: valor;
}

Seletor: Indica qual elemento será estilizado.
Propriedade: O que será alterado (ex: cor, tamanho, fonte).
Valor: O valor atribuído à propriedade.
