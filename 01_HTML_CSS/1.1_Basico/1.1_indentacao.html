O que é Indentação?

Definição

Indentação é o espaçamento aplicado no início das linhas de código para mostrar a hierarquia dos elementos.
No HTML, a indentação não afeta o resultado visual da página, mas torna o código mais legível e profissional.

Exemplo Visual

<ul>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
        </ul>
  • Item 1
  • Item 2
  • Item 3
