O que é CSS?

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata).

Como usar CSS?

  • Inline: Estilo aplicado diretamente no elemento.
  • Interno: Estilo definido dentro da tag <style> no <head>.
  • Externo: Estilo definido em um arquivo .css separado.
