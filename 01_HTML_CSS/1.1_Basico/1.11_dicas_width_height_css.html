Dicas sobre Width/Height

Boas práticas para largura e altura.

Exemplo Visual

width: 50%;
min-width: 150px; max-width: 300px;
box-sizing: border-box;

Dicas

  • Use porcentagem para layouts responsivos.
  • Defina min-width e max-width para limitar tamanhos.
  • Utilize box-sizing: border-box para facilitar o controle do layout.
