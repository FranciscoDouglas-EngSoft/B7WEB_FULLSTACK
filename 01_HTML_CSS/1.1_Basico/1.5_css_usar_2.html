Como funciona o CSS?

Veja exemplos de seletores e comentários no CSS.

Seletores

  • #id: Seleciona um elemento pelo atributo id.
  • .classe: Seleciona todos os elementos com a mesma classe.
  • elemento: Seleciona todas as tags de um tipo, como p ou h1.

Comentários no CSS

Comentários ajudam a documentar o código e não são exibidos no navegador.

/* Este é um comentário em CSS */
body {
    background: #f4f4f4;
}
