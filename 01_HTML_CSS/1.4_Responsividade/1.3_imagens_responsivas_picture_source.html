Imagens Responsivas: picture e source

Utilize <picture> e <source> para servir imagens diferentes conforme o tamanho da tela.

Exemplo Visual

Redimensione a janela para ver a troca de imagem.

Resumo

  • <picture> permite múltiplas fontes de imagem.
  • <source> define condições para cada imagem.
  • O <img> é o fallback.
