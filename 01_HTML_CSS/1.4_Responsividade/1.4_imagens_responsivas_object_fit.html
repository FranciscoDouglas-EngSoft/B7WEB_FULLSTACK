Imagens Responsivas: object-fit

Use object-fit para controlar o redimensionamento de imagens.

Exemplo Visual

  • Cover: Preenche o container, pode cortar a imagem.
  • Contain: Ajusta a imagem para caber inteira no container.

Resumo

  • object-fit: cover mantém o aspecto e cobre todo o espaço.
  • object-fit: contain mantém o aspecto e mostra a imagem inteira.
