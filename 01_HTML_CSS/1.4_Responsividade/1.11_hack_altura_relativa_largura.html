Hack CSS: Altura relativa à largura

Use aspect-ratio ou padding para criar elementos com altura proporcional à largura.

Exemplo Visual

Aspect-ratio 16:9
Padding-top 56.25% (16:9)

Resumo

  • aspect-ratio define proporção diretamente (moderno).
  • padding-top com porcentagem simula altura relativa à largura (compatível).
