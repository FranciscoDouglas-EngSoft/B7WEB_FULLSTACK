Formatação de textos no CSS (1/2)

Principais propriedades para estilizar textos.

Exemplo Visual

Cor do texto (color: #d32f2f;)

Tamanho da fonte (font-size: 24px;)

Família da fonte (font-family: 'Courier New', Courier, monospace;)

Peso da fonte (font-weight: bold;)

Estilo da fonte (font-style: italic;)

Texto centralizado (text-align: center;)

Propriedades

  • color: Cor do texto.
  • font-size: Tamanho da fonte.
  • font-family: Família da fonte.
  • font-weight: Peso (negrito).
  • font-style: Estilo (itálico).
  • text-align: Alinhamento do texto.
