Formatação de textos no CSS (2/2)

Propriedades avançadas para estilizar textos.

Exemplo Visual

Maiúsculas (text-transform: uppercase;)

Sublinhado (text-decoration: underline;)

Espaçamento entre letras (letter-spacing: 4px;)

Altura da linha (line-height: 2;)

Espaçamento entre palavras (word-spacing: 16px;)

Propriedades

  • text-transform: Transforma o texto (maiúsculas/minúsculas).
  • text-decoration: Decoração (sublinhado, linha sobreposta, etc).
  • letter-spacing: Espaçamento entre letras.
  • line-height: Altura da linha.
  • word-spacing: Espaçamento entre palavras.
